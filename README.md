Игра "морской бой". Каждый игрок расставляет на своем поле корабли, задача игроков - своими ходами уничтожить все вражеские корабли. Для запуска проекта:

Склонируйте репозиторий.
Выполните команды:
1) cd sea_battle
2) mkdir build
3) cd build
4) cmake ..
5) make
6) ./tp_sea_battle
