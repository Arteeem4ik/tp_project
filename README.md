Sea battle game. Each player places ships on his field, the task of the players is to destroy all enemy ships with their moves. To start a project:

Clone the repository.
Run the commands:
1) cd sea_battle
2) mkdir build
3) cd build
4) cmake ..
5) make
6) ./tp_sea_battle
